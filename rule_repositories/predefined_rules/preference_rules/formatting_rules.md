# Formatting and Style Rules

## Overview

These are Tier 3 Preference Rules that provide formatting suggestions and style guidelines. These rules are adaptive and user-override is allowed.

## Rule: Document Structure

**ID:** preference_document_structure  
**Tier:** Preference (Tier 3)  
**Enforcement:** Adaptive  
**Severity:** Low

### Description

Suggests improvements to document structure and formatting for better readability.

### Patterns

**Suggestions:**
- Use headers for section organization
- Add bullet points for lists
- Include proper spacing
- Use consistent formatting

### Enforcement

- Action: SUGGESTION
- Auto-block: No
- User Override: Yes
